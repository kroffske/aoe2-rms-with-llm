# Command Reference

Active commands.

## By Category

### work/* - Work Modes
| Command | Description |
|---------|-------------|
| `/work:lead` | Direct implementation mode |
| `/work:team` | Delegate to agents |
| `/work:plan` | Planning with feedback |

### task/* - Epic Management
| Command | Description |
|---------|-------------|
| `/task:create` | Create new epic |
| `/task:implement` | Execute task from epic |
| `/task:resume` | Continue existing epic |
| `/task:close` | Close epic |

### dev/* - Code Quality
| Command | Description |
|---------|-------------|
| `/dev:lint` | Run quality gates |

### release/* - Git Workflow
| Command | Description |
|---------|-------------|
| `/release:commit` | Conventional commit |
| `/release:push` | Full release |

### gh/* - GitHub Integration
| Command | Description |
|---------|-------------|
| `/gh:projects` | Init GitHub tracker |

### Utilities
| Command | Description |
|---------|-------------|
| `/reflection` | Analyze work process |
| `/sync` | Sync .claude to projects |
